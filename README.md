# 🐾 MyPocketPet

Виртуальный питомец нового поколения — тамагочи с тактическими боями, эмоциями, магазином, ежедневными заданиями и социальным взаимодействием между игроками.

**🌐 Демо:** [tamaguchi.onrender.com](https://tamaguchi.onrender.com)

---

## 🎮 Возможности

### Питомец
- **6 персонажей** — Фиалка, Сакура, Листик, Тыковка, Снежок, Уголёк
- **6 действий** — кормить, играть, спать, учить, гулять, музыка
- **4 навыка** — творчество, интеллект, ловкость, социальность
- **60 уровней** с уникальными названиями и экспоненциальной прокачкой
- **Эволюция** — визуальные стадии на уровнях 20 / 40 / 60 (искры → крылья → корона)
- **Настроение по времени суток** — утром сонный, днём активный, ночью хочет спать
- **AI-чат** — разговор с питомцем, личность зависит от характера

### Мини-игры
- **🧱 Тетрис** — свайп-управление, награда монетами и XP
- **🎁 Сундуки** — 9 сундуков, 3 сокровища, 6 бомб, кулдаун 6 часов
- **⚔️ Арена** — тактические PvE бои с системой эмоций (страх, злость, усталость, храбрость, спокойствие)

### Экономика
- **🪙 Монеты** — зарабатывай в играх, трать на действия и магазин
- **🛍️ Магазин** — шапки (6шт), фоны (5шт), декор (5шт) — всё отображается на питомце
- **📋 Ежедневные задания** — 5 квестов с наградами, обновляются каждый день
- **🏅 Достижения** — 10 ачивок от «Первое кормление» до «Чемпион»

### Социальное
- **💬 Глобальный чат** — общение между всеми игроками в реальном времени
- **🌍 Список игроков** — просмотр всех питомцев, посещение профилей
- **⚔️ PvP бои** — вызов на бой из лидерборда, автоматический расчёт
- **🎁 Подарки** — отправляй подарки и монеты другим игрокам
- **🏆 Лидерборд** — топ-10 по XP

## 🗄 База данных

**7 таблиц:**

- `users` — Telegram пользователи (tg_id, имя, аватар)
- `session` — серверные сессии (connect-pg-simple)
- `pets` — питомцы (статы, навыки, трейты, монеты)
- `daily_stats` — ежедневная статистика действий
- `chat_history` — история чата с питомцем
- `pvp_battles` — PvP вызовы и результаты
- `gifts` — подарки между игроками
- `global_chat` — глобальный чат

## ⚙️ Установка и запуск

### Переменные окружения

```env
DATABASE_URL=postgresql://user:pass@host:5432/dbname
BOT_TOKEN=123456:ABC-DEF           # от @BotFather
BOT_USERNAME=MyPocketPetBot        # имя бота без @
SESSION_SECRET=random_secret_key
NODE_ENV=production
```

### Локально

```bash
git clone https://github.com/your-repo/mypocketpet.git
cd mypocketpet
npm install
# Создай .env с переменными выше
node server.js
# Открой http://localhost:3000
```

### Деплой на Render

1. Создай **Web Service** → подключи GitHub репозиторий
2. Build: `npm install`
3. Start: `node server.js`
4. Добавь Environment Variables
5. Создай **PostgreSQL** базу, скопируй `DATABASE_URL`
6. В BotFather: `/setdomain` → `tamaguchi.onrender.com`
